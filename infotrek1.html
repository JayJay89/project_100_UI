CSS Property: Margin and Padding

  • Margin can accept negative values
  • Padding cannot accept negative values
Try This

  • Set the margin to a negative value to see what it looks like.

CSS Property: Width and Height

  • Height and width can be given fixed or percentage values
  • Percentage values are calculated based on the parents dimensions
  • Height, when given percentage value, only inherit from a direct parent
height: 300px;
width: 300px;
height: 50%;
width: 50%;
height: 20%;
width: 20%;
Try This
  • Give .info-box-give-me-height height: 500px
  • Give .info-box-give-me-height height: 100%
  • Drag the last box out of .info-box-give-me-height

CSS: Margin Collapse

Top and bottom margins of blocks are sometimes combined (collapsed) into a single margin whose size is the largest of the margins combined into it, a behavior known as margin collapsing.

When there are 2 block elements arrange next to each other, margins collapses

Margin Collapse
Margin Collapse

How do we prevent this?

Try This
  • Change both boxes(box-margin-collapse) display: value to inline-box and give them a 100% width
